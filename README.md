# eAud

## Programa responsável por buscar os dados da API do e-Aud

## Virtual env

### pip install virtualenv venv

### venv\Scripts\activate

### pip install -r requirements.txt

## Para transformar o código em um executável do windows, deve-se instalar a blibioteca

### pip install pyinstaller e executar o seguinte comando para realizar a compilação

### pyinstaller --onefile main.py

### O arquivo executável estará na pasta criada pelo programa DIST
